DEPLOY YOUR OWN SANCTUARY

1. Create a Cloudflare Account.

2. Clone: git clone https://github.com/Creative-Workz-Studio-LLC/cpisi-template

3. Run: npx wrangler secret put GEMINI_API_KEY (Provide your key from Google AI Studio).

4. Run: npx wrangler deploy

5. Point this UI to your new Worker URL in Settings.
