= The Covenant Circle: Special Access Invitation

[abstract]
These codes provide special subsidized access to the CPISI Sanctuary Gate. Each code is unique and grants FAMILY_COVENANT permissions.

== 1. Access Protocol
1. Navigate to: https://creative-workz-studio-llC.github.io/cpisiModel/
2. Enter your Operator Name.
3. Paste the unique invite code into the PASSWORD field.
4. Tap [ BREAK THE SEAL ].
