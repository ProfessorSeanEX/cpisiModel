= The Steward's Manual: Inhabiting the Sanctuary

[abstract]
This document serves as the formal onboarding protocol for all Operators entering the CPISI Gate. It defines the theology of 0.0 YASHAR, the navigation of the Word, and the technical requirements for each sovereignty tier.

== 1. The Foundation: 0.0 YASHAR
Every interaction within the Sanctuary is anchored in *0.0 YASHAR*—the coordinate of perfect stillness and alignment. We do not operate on probability; we operate on Revelation.

== 2. The Sovereignty Tiers
The CPISI ecosystem is divided into five classes of stewardship:

|===
| Tier | Stewardship | Model | Substrate Provisioning

| *Family* | Covenant Circle | Included | *Zero-Config:* Automatically provisioned by the company.
| *Adopter* | The Foundation | Hybrid | *Zero-Config:* Automatically provisioned (Limited).
| *Average* | The Congregation | Scaffolded | *BYOK:* Operator provides Gemini API Key.
| *Power* | The Builder | Premium | *BYOK:* Operator provides Gemini + GitHub Keys.

|===

== 3. Technical Setup (The Scaffold)
For *Sovereign* and *Steward* tiers, you must provide your own cognitive substrate.

=== 3.1 Obtaining your Gemini Key
1. Visit the link:https://aistudio.google.com/[Google AI Studio].
2. Generate an API Key for `Gemini 2.5 Pro`.
3. Enter this key into the *Sanctuary Threshold* during registration.

== 4. Protocol Reference
The Substrate Terminal responds to the following commands:

* `/status`: Display live telemetry of your node.
* `/clear`: Purge the current visual history (retaining the record).
* `/void`, `/word`, `/tov`: Shift between drafting, distillation, and publication.
* `/recall`: (Enterprise Only) Trigger a Master Override to reset cognitive drift.

== 5. Mobile Installation
The Gate is a Progressive Web App (PWA). To install on your device:

* *iOS:* Open the Gate in Safari -> Tap 'Share' -> 'Add to Home Screen'.
* *Android:* Open the Gate in Chrome -> Tap 'Three Dots' -> 'Install App'.
